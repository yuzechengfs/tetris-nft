# 🎮 Tetris NFT

Tetris NFT is an Android game built on **Solana Mobile Seeker**, combining classic Tetris gameplay with blockchain technology.

Players can capture their best in-game moments and mint them as NFTs on the Solana blockchain — making gameplay achievements collectible, ownable, and shareable.

---

## ✨ Features

- 🧱 Classic Tetris gameplay on Android
- 📸 Capture gameplay highlights as screenshots
- 🪙 Mint screenshots as NFTs on Solana
- 📱 Optimized for Solana Mobile Seeker
- 🔐 User-owned digital assets

---

## 🚀 How It Works

1. Play Tetris and achieve high-score or interesting moments  
2. Capture a screenshot inside the app  
3. Mint the screenshot as an NFT  
4. Store and showcase your NFT in your wallet  

---

## 🔗 Blockchain

- Network: Solana
- Wallet: Compatible with Solana Mobile Stack wallets
- NFTs: Minted directly by the user

---

Seeker App store only
